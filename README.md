# 🚀 Uni-Voyager — Your AI Career Mentor

CareerCompass AI is an AI-powered career guidance platform that helps users navigate their professional journey using intelligent recommendations, resume analysis, and an interactive AI mentor built with IBM Watson AI.

---

## Overview

Choosing the right career path can be overwhelming. CareerCompass AI simplifies this process by analyzing user skills, experience, and goals to provide personalized, data-driven career insights.

The platform includes an AI mentor that users can interact with in real time to get guidance on careers, skills, and job preparation.

---

##  Features

###  AI Career Mentor (Watson AI)

* Chat-based AI mentor for career advice
* Answers questions about roles, skills, and growth paths
* Provides personalized recommendations

### Resume Analysis

* Evaluates resumes for strengths and weaknesses
* Suggests improvements for ATS optimization
* Identifies missing skills

### Career Recommendations

* Suggests career paths based on user profile
* Aligns skills with industry demand

### Skill Gap Analysis

* Highlights missing or weak skills
* Recommends what to learn next

### Learning Roadmap

* Step-by-step plan to reach target roles
* Structured progression for skill development

## ⚙️ How It Works

1. User inputs skills, interests, or uploads a resume
2. AI processes the data using Watson AI
3. System analyzes career fit and skill gaps
4. User receives:

   * Career recommendations
   * Resume feedback
   * Learning roadmap
5. User interacts with AI mentor for guidance

## 🚀 Getting Started

### Prerequisites

* Python 3.x
* pip

### Installation

```bash
git clone https://github.com/mtafoya-web/CareerCompass_AI.git
cd CareerCompass_AI
pip install -r requirements.txt
```

### Run the App

```bash
python app.py
```

Then open:

```
http://127.0.0.1:5000
```
